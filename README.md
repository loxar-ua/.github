# .github
Цей відкритий репозиторій створенний для того, щоб познайомити зацікавленних людей з нашим студентським проєктом: платформи моніторингу публічних справ "Шкандаль"

**Формулювання проблеми**
Сучасне українське суспільство перебуває в умовах інформаційного перевантаження. Щоденний потік новин оновлюється щохвилини, ускладнюючи фокусування на критично важливих подіях. Інформація про резонансні публічні справи швидко втрачає видимість, "тонучи" у нових інфоприводах. Повномасштабна війна лише загострила цю проблему, згенерувавши ще більший обсяг новин та посиливши "синдром втраченої пам'яті" у суспільства.

**Проблема**
Через постійний, неструктурований потік новин увага громадськості та медіа швидко розсіюється. Важливі справи (наприклад, корупційні розслідування) залишаються без належного моніторингу, оскільки нові, "гарячіші" публікації витісняють їх з порядку денного.
Як наслідок, у громадян відсутній зручний інструмент, щоб відстежити повну хронологію ключових подій - від першої публікації до фінального рішення (наприклад, вироку суду чи звільнення посадовця).

**Користувачі**
Активні громадяни: Українці з активною позицією, які прагнуть контролювати дії влади та стежити за державними процесами.
Професійна спільнота: Журналісти, громадські активісти, аналітики, юристи.
Ці користувачі зацікавлені у моніторингу публічних справ, але не мають часу чи ресурсів для щоденного відстеження десятків паралельних подій, що тривають місяцями чи роками.
Потреба користувача: Їм потрібна єдина платформа, яка дозволить зручно моніторити перебіг публічних справ, бачити їхню повну хронологію та кінцеві результати.

**Наслідки проблеми**
Відсутність структурованого та доступного відстеження подій у часі призводить до системних негативних наслідків для суспільства:
* Безкарність: Винні у правопорушеннях уникають відповідальності, оскільки суспільний запит на справедливість згасає разом із "гайпом" новини.
* Ерозія громадського контролю: Суспільство втрачає здатність ефективно контролювати владу та вимагати завершення справ.
* Руйнація інституту репутації: Без довгострокової пам'яті про вчинки, неможливо сформувати інститут публічної репутації, де посадовці відповідають за свої дії.
* Гальмування змін: Важливі суспільні трансформації (наприклад, реформи) затягуються або не відбуваються, оскільки проблеми не отримують постійної уваги.
